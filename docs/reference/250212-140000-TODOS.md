## 1. 스타일 커스텀 vs 컴포넌트 주입 분리

- **현재 문제**: `LoginForm` 내부에서 `components` 유무로 분기 처리하고 있음. 스타일만 쓰는 컴포넌트와 주입용 컴포넌트를 하나로 섞어둔 상태.
- **목표**:
  - **스타일 커스텀 전용**: `LoginForm`, `RegisterForm`, `WorkspaceJoinForm` 등 — `className` / `style` / `*ClassName` / `*Style` 만 받고, 내부 마크업은 고정. 컴포넌트 주입 prop 없음.
  - **컴포넌트 주입 전용**: `LoginFormCustom`, `RegisterFormCustom` 등 — `components` 로 Container, Card, Input, Button 등 UI 요소만 주입. 동일 로직은 훅(`useLoginForm` 등)으로 공유.
- **작업**:
  - [ ] `LoginForm`에서 `components` prop 및 주입 분기 제거 (이전 스타일 전용만 남김).
  - [ ] `LoginForm` 스펙에서 "custom components when components prop is provided" 테스트 제거 또는 `LoginFormCustom` 테스트로 이전.
